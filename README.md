```bash

npm install

# Start production server
npm start
```

## Testing

```bash
# Run E2E tests
npm run test:e2e

# Run tests with UI
npm run test:e2e:ui

# Run tests with coverage
npm run test:coverage

# View test report
npm run test:e2e:report
```
